The Flutter SDK is the official client integration surface within the Sankofa ecosystem. [View source on GitHub](https://github.com/Sankofa-HQ/sankofa_sdk_flutter).
It provides:

- queued event delivery
- automatic flush behavior
- device and network context
- identity stitching
- people profile updates
- **session replay** (wireframe & screenshot)

## Identity and People

To connect anonymous data to a known user, use `identify`. To set profile attributes used in the dashboard, use `setPerson`.

```dart
// 1. Identify the user (Internal ID)
await Sankofa.instance.identify("user_987");

// 2. Set profile traits (Display Name, Email, Avatar)
await Sankofa.instance.setPerson(
  name: "John Doe",
  email: "john@example.com",
  avatar: "https://example.com/avatars/john.jpg",
  properties: {
    "plan": "premium"
  }
);
```

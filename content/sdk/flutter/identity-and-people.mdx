## Identify a known user

```dart
await Sankofa.instance.identify('user_123');
```

When the user changes from the anonymous ID to a known ID, the SDK queues an
alias transport item that links the previous ID to the new one.

## Reset on logout

```dart
await Sankofa.instance.reset();
```

Reset clears the current user identity, creates a new anonymous ID, and creates
a new session ID.

## Set people properties

```dart
await Sankofa.instance.peopleSet({
  'plan': 'growth',
  'company_name': 'ThrivnMe',
  'role': 'operator',
});
```

People updates use the `people` transport and target the current `distinct_id`.

| --- | --- |

| `peopleSet()` | `POST /api/v1/people` |

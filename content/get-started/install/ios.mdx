The Sankofa iOS SDK is a native Swift library with Objective-C interop. It includes six Sankofa products (Analytics, Catch, Switch, Config, Pulse, Replay) and uses GRDB for thread-safe SQLite-backed offline queueing. Apple Privacy Manifest support is built in.

OTA Deploy is not exposed on iOS — for hybrid apps that need OTA, use the React Native SDK.

## Requirements

- **Xcode 15+** (Swift 5.9+)
- **iOS 13+** deployment target
- **Privacy Manifest** support (Apple 2024 requirement) — bundled with the SDK

## 1. Add the SDK

<Tabs defaultValue="spm">
  <Tab value="spm" label="Swift Package Manager">
    In Xcode:

    1. Open **File → Add Package Dependencies…**
    2. Enter the URL: `https://github.com/sankofa-hq/sankofa-ios`
    3. Pick **Up to Next Major Version** from `0.1.0`.
    4. Add the `SankofaIOS` library to your app target.

    Or pin in `Package.swift`:

    <CodeBlock lang="swift" filename="Package.swift" code={`.package(url: "https://github.com/sankofa-hq/sankofa-ios.git", from: "0.1.0")`} />
  </Tab>
  <Tab value="cocoapods" label="CocoaPods">
    <CodeBlock lang="ruby" filename="Podfile" code={`platform :ios, '13.0'

target 'YourApp' do
  use_frameworks!
  pod 'SankofaIOS', '~> 0.1'
end`} />

    <CodeBlock lang="bash" code={`pod install`} />

    Note: the Swift target name is `SankofaIOS`. Import with `import SankofaIOS`.
  </Tab>
</Tabs>

## 2. Initialize at app launch

Initialize early — in `@main`'s `init()` or `application(_:didFinishLaunchingWithOptions:)` — so device metadata flows into every event from the first frame.

<CodeGroup>
<CodeGroupItem tab="ios" label="Swift">
<CodeBlock lang="swift" filename="App.swift" code={`import SwiftUI
import SankofaIOS

@main
struct YourApp: App {
    init() {
        Sankofa.shared.initialize(
            apiKey: ProcessInfo.processInfo.environment["SANKOFA_KEY"] ?? "",
            config: SankofaConfig(
                endpoint: "https://api.sankofa.dev",
                recordSessions: true,
                maskAllInputs: true
            )
        )
    }

    var body: some Scene {
        WindowGroup { ContentView() }
    }
}`} />
</CodeGroupItem>
<CodeGroupItem tab="ios-objc" label="Objective-C">
<CodeBlock lang="objective-c" filename="AppDelegate.m" code={`#import <SankofaIOS/SankofaIOS-Swift.h>

- (BOOL)application:(UIApplication *)application
        didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
    SankofaConfig *config = [[SankofaConfig alloc] initWithEndpoint:@"https://api.sankofa.dev"
                                                              debug:NO
                                                trackLifecycleEvents:YES
                                                flushIntervalSeconds:30
                                                          batchSize:50
                                                     recordSessions:YES
                                                      maskAllInputs:YES];
    [[Sankofa shared] initializeWithApiKey:@"sk_live_..." config:config];
    return YES;
}`} />
</CodeGroupItem>
</CodeGroup>

### Common config options

<ParameterTable
  params={[
    { name: "endpoint", type: "String", default: "https://api.sankofa.dev", description: "Server base URL." },
    { name: "recordSessions", type: "Bool", default: "true", description: "Enable session replay (screenshot mode)." },
    { name: "maskAllInputs", type: "Bool", default: "true", description: "Automatically mask all UITextField/UITextView views in replays." },
    { name: "debug", type: "Bool", default: "false", description: "Verbose console logging during development." },
    { name: "trackLifecycleEvents", type: "Bool", default: "true", description: "Auto-track $app_opened, foreground/background." },
    { name: "flushIntervalSeconds", type: "TimeInterval", default: "30", description: "How often the queued events are uploaded while the app is foregrounded." },
    { name: "batchSize", type: "Int", default: "50", description: "Maximum events per upload batch." },
    { name: "captureScale", type: "CGFloat", default: "0.35", description: "Replay screenshot resolution. Lower = smaller files." },
  ]}
/>

## 3. Verify the install

<CodeBlock lang="swift" code={`Sankofa.shared.track("install_check", properties: ["from": "ios-quickstart"])`} />

Build and run in the simulator or on a device. Open [app.sankofa.dev](https://app.sankofa.dev) → **Live events** — the event should appear with an `ios` source label within a few seconds.

<Callout tone="tip" title="Privacy Manifest is included">
The SDK bundles its `PrivacyInfo.xcprivacy` declaration. App Store Connect will surface it in the privacy report alongside your own. No extra action required.
</Callout>

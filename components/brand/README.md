# Brand components

These components are **synced from `website-v2`** to keep the docs site visually identical to the marketing site.

| File | Sourced from |
|---|---|
| `SankofaMark.tsx` | [`website-v2/components/SankofaMark.tsx`](../../../website-v2/components/SankofaMark.tsx) — verbatim copy |
| `DocsBadge.tsx`   | Docs-specific variant of the "OS" chip in [`website-v2/components/Nav.tsx`](../../../website-v2/components/Nav.tsx) |

If you change anything in this directory, change the corresponding file in `website-v2` too. The two sites diverging visually erodes brand trust — keep them in sync until both have stabilized for at least three months, after which we promote shared components to a `@sankofa/ui` package.
